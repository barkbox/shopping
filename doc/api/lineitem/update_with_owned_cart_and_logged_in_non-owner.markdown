# LineItem API

## Update with owned cart and logged in non-owner

### PATCH /line_items/:id

### Parameters

| Name | Description | Required | Scope |
|------|-------------|----------|-------|
| id | Line item id | true |  |

#### Headers

<pre>Content-Type: application/vnd.api+json

<pre>{"data":{"id":5,"type":"line_items","attributes":{"quantity":2}}}</pre>
